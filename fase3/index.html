Propósito del sitio:

El propósito principal de este sitio Web es dar a conocer la estructura básica HTML, mediante sus distintas etiquetas, elementos y atributos mediante los cuales se podrá dar una estructura coherente teniendo en cuenta las normas semánticas. También se dará el aprendizaje de las distintas formas de diseño que podrá darle al sitio Web mediante el uso de Flexbox y CSS Grid.
